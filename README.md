## initx-plugin-starter

`initx` plugin starter template.

## Usage

```bash
npx initx plugin add starter
```

try the first command

```bash
npx initx start
```

## Development

Run stub

```bash
pnpm stub
```

Install plugin from current directory

```bash
npx initx plugin add .
```

Enjoy your plugin development!
